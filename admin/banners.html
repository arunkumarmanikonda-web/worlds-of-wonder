📖 Developer Notes

Video banners: Upload your MP4 video to a CDN or public storage, then paste the URL in the Video URL field. The video will autoplay muted and loop.


Image banners: Paste any public image URL (JPG/PNG/WebP). Recommended minimum size: 1920×1080px.


Gradient fallback: If no media URL is provided, the CSS gradient is used as the background. Edit the gradient string using CSS linear-gradient syntax.


localStorage override: This admin panel saves to localStorage['wow_banners_override']. To use the JSON file config instead, click Reset to Defaults.
